# 04 · 网格守卫最少放置问题

**课程**：算法设计与分析  
**学期**：大二下学期  
**问题**：m×n 矩形网格，每个格子可放一名守卫（覆盖自身 + 上下左右十字形），求最少守卫数及方案数。等价于图论**最小支配集**问题，NP-困难。

## 剪枝策略一览

| 剪枝名称 | 作用 |
|---------|------|
| 最优性剪枝 | 当前卫兵数 ≥ 最优解 → 返回 |
| 不可达剪枝 | 存在未覆盖格子且无剩余候选 → 返回 |
| 后缀覆盖剪枝 | 剩余全部放也不能全覆盖 → 返回 |
| 下界估计 | `need ≥ (未覆盖数 + 4) / 5` |
| 必须放置剪枝 | 当前位置不被后续覆盖 → 强制放 |
| 搜索顺序优化 | 高收益分支优先探索 |
