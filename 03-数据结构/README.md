# 03 · 数据结构（C 语言实现）

**课程**：数据结构  
**学期**：大二上学期  
**内容**：C 语言实现线性表、数组 ADT、稀疏矩阵、栈、树

## 文件索引

| # | 文件 | 数据结构 | 核心操作 |
|---|------|---------|---------|
| 01 | `01-顺序表-静态数组实现.c` | 顺序表（静态） | Init / Create / Insert / Delete / Locate |
| 02 | `02-顺序表-动态分配实现.c` | 顺序表（动态） | malloc 分配, 选择排序, 两表合并 |
| 03 | `03-顺序表-合并与排序.c` | 顺序表（合并排序） | 归并 + 选择排序 + 插入 / 删除 / 查找 |

| 05 | `05-稀疏矩阵三元组.c` | 稀疏矩阵 | 基本转置 O(nu×tu) vs 快速转置 O(nu+tu) |
| 06 | `06-字符串输入输出.c` | 字符串 | 输入输出基础 |
| 07 | `07-栈序列判定.c` | 栈 | 模拟出入栈判定合法性 |
