##c++
注释
1.单行注释:形式:  //+注释内容

2.多行注释:形式: /*+注释内容+*/

开头格式
1.#include<iostream>   引入输入输出流头文件

2.using namespace std; 使用标准命名空间

3.int main(){}         程序入口函数

4.cout<<"hello world"<<endl;输出语句
cout：全称是 “console output”，即 “控制台输出”，是标准输出流对象，用来向屏幕打印内容。

<<：是 “流插入运算符”，作用是把右边的内容 “插入” 到 cout 流中，也就是输出到屏幕。
"hello world"：是字符串常量，会原样输出到屏幕。

endl：全称是 “end line”，即 “换行”，作用有两个：① 输出一个换行符（相当于按回车键）；② 刷新输出缓冲区，确保内容立刻显示在屏幕上。

分号 ;：C++ 中语句的结束标志，必须加，否则编译报错。

5.return 0;     main函数返回值为0(可选但推荐)

6.system("pause");  让程序运行结束后暂停,等待用户按任意键后退出窗口;
system是定义在c++标准库中的一个标准函数(stdlib.h头文件)，pause是一个参数，表示暂停程序的执行，等待用户按任意键后继续执行下一条语句。

关键字
变量
1.变量的定义: 变量类型 变量名

2.作用:给一段指定的内存空间取名,方便操作内存

3.初始化:变量类型 变量名=初始值;

常量
1.定义方法 #define 常量名 常量值;const 数据类型 常量名=常量值;

整型常量: 123,456,0,-123等;
数据类型:
short :2字节  -2^15~2^15-1
int   :4字节  -2^31~2^31-1
long  :4字节(win) 8字节(linux)  -2^31~2^31-1
long long int:8字节  -2^63~2^63-1

浮点常量: 3.14,0.123,-1.23等;
float :4字节 7位有效数字 -3.4E+38~3.4E+38
double:8字节 15位有效数字 -1.7E+308~1.7E+308
float定义时,必须在常量后加字符f或F,否则默认为double类型,浪费内存空间.
默认情况下输出一个小数时会保留六位有效数字,可使用cout.precision(n)设置有效数字位数,n为整数

3e2=3*10^2=300

sizeof运算符:
sizeof(数据类型)  sizeof(变量名)  sizeof(常量值)
作用:返回数据类型或变量名占用的内存空间大小(单位:字节)
sizeof(sizeof(int))=4;

字符型:
作用:用于显示单个字符
语法:char 变量名='字符'
c和c++中字符变量值占据一个字节
字符变量不是把字符放入内存,而是把对应的ascll编码的之放入储存空间
输出方法:cout<<变量名<<endl;
a~z:97~122
A~Z:65~90
