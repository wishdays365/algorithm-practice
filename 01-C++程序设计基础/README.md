# 01 · C/C++ 程序设计基础

**课程**：C/C++ 程序设计  
**学期**：大一上学期  
**内容**：从零开始学习 C/C++ 基础语法

## 文件索引

| 文件 | 内容 | 知识点 |
|------|------|--------|
| `01-基础输入输出.cpp` | Hello World | cin / cout, 变量, using namespace |
| `02-字符串与向量.cpp` | string 和 vector 入门 | size(), push_back() |
| `03-输入解析.cpp` | 多行输入处理 | getline(), 流状态控制 |

| `07-字符串分割.cpp` | 按分隔符拆分 | substr() + find() |
| `08-格式化解析.cpp` | 结构化数据提取 | stringstream 读写综合 |
