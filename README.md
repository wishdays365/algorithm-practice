## 亮点项目

### 网格守卫最少放置问题

NP-困难的图最小支配集问题，完整实现回溯到分支限界的演进：

- **6 种剪枝策略**渐进增强
- 10×10 网格分支限界比回溯快 **2.78 倍**（91s vs 255s）
- 含完整实验报告（Mermaid 流程图、甘特图、可视化）

### 数值计算工具集

`11-数值计算工具集.cpp` 包含 18 种数值方法，覆盖期末考试全部知识点。
